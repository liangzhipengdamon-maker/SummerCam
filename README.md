# 2026中山大学人工智能育苗夏令营宣发物料库

本仓库用于管理 **2026中山大学人工智能育苗夏令营** 的招生宣传与转化物料，作为后续小红书图文、朋友圈文案、海报文案、家长群话术、FAQ、课程顾问转化话术、渠道合作资料和设计交付说明的统一资料源。

## 核心传播定位

> **全球前2%顶尖科学家任课，带初高中生完成真实AI项目。**

辅助表达：

- 中山大学珠海校区 5天4晚沉浸式学习
- 面向初一至高三学生的AI启蒙与项目实践课程
- 大模型体验 + Python编程 + 机器学习入门 + AI Agent项目实践
- 从创意构思、项目分工、研发实践到成果展示与专家评审
- 完成课程可获得写实性结业证书，优秀项目有机会获得项目优秀证书

## 项目基础信息

- **项目名称：** 2026中山大学人工智能育苗夏令营
- **项目地点：** 中山大学珠海校区，广东省珠海市唐家湾镇大学路2号
- **招生对象：** 初一至高三学生，13-18岁，可分组别上课
- **课程周期：** 5天4晚
- **课程费用：** 6800元/人
- **第一期：** 7月19日—7月23日
- **第二期：** 7月26日—7月30日
- **咨询报名：** 以正式发布物料中的报名方式为准
